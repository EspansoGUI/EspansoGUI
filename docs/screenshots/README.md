## Screenshot Guidelines

1. **Redaction**: All personal paths (e.g., `C:\Users\username`) should be redacted or use placeholder paths
2. **Snippet Content**: Use generic example snippets, not personal data
3. **Resolution**: Capture at 1920x1080 or similar standard resolution
4. **Theme**: Capture in dark theme (default) for consistency
5. **State**: Show views with representative data loaded

## Capturing Screenshots

On Windows:
```
Win + Shift + S (Snipping Tool)
```

Or use the GUI's built-in screenshot capability if available.
